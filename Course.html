    My Course

    Introduces My Alogunuin College Computer Programming.

    CST8116 : Introduction to Computer Programming

    Level 1

    This course offers a comprehensive introduction to logic, problem-solving, and programming, focusing on Java as the programming language. Students will develop essential skills in algorithmic thinking, coding conventions, and object-oriented programming, supported by hands-on lab assessments to reinforce practical application.

    CST8118 : Computer Essentials

    Level 1

    This course covers computer maintenance, operating systems, productivity software, and environmental sustainability in Information Technology. Students learn about hardware, secure systems, and use spreadsheets for problem-solving. They also discuss the financial and environmental impact of computer technology. Practical assessments include virtualization software usage.

    CST8215 : Introduction to Database

    Level 1

    Databases are used to store data and are a core component of many information technology systems. Students learn the fundamentals of relational databases design using Entity Relation Diagrams (ERDs), and use Structured Query Language (SQL) to create, modify and query a database. Students design and create databases that are maintainable, secure and adaptable to change in business requirements, using normalization. Students become familiar with the functions of a Database Management System (DBMS) and its components in comparison with legacy systems and alternative information storage mechanisms.

    CST8215 : Introduction to Database

    Level 2

    This course explores database systems, automation of data processing tasks, and their connection to IT security. Students gain practical experience with Oracle and MySQL databases, learning advanced SQL, modeling tools, and programming with PL/SQL. Topics covered include data storage, retrieval, security, and data warehousing.

    CST8102 : Operating System Fundamentals (GNU/Linux)

    Level 2

    This course provides a comprehensive exploration of operating systems, including their basic concepts, components, and interactions with hardware and software, with a focus on GNU/Linux. Students gain theoretical knowledge and practical skills in operating system structures, process management, storage management, installation, configuration, and administration.

    CST8284 : Object Oriented Programming (Java)

    Level 2

    This course explores Java-based Object-Oriented Programming (OOP), covering essential concepts like encapsulation, inheritance, abstraction, and polymorphism. Students also gain exposure to data structures, algorithms, and basic Graphical User Interface (GUI) programming.

    CST8285 : Web Programming

    Level 2

    The World Wide Web (WWW) has become an integrated part of everyday life. Students develop basic skills of web programming, website design and implementation. JavaScript, HTML5, and PHP are used to explore web-based solutions to problems of increasing interactivity and complexity. Lectures are reinforced by practical assignments that encourage students to construct and maintain their own websites.
